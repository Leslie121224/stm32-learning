## 🛠 ADXL345 與 Nucleo-F401RE 硬體接線表 (I2C 模式)

| ADXL345 引腳 | Nucleo 引腳 | 功能說明 |
| :--- | :--- | :--- |
| **VCC** | **3.3V** | 電源輸入 (Power Supply) |
| **GND** | **GND** | 接地 (Ground) |
| **SCL** | **PB8** | I2C 時脈線 (I2C1_SCL) |
| **SDA** | **PB9** | I2C 資料線 (I2C1_SDA) |
| **CS** | **3.3V** | **關鍵：接高電位 (High) 進入 I2C 模式** |
| **SDO** | **GND** | **決定地址：接 GND 時，7-bit 地址為 `0x53`** |

> **📌 專業筆記：**
> * **I2C 模式切換**：ADXL345 預設支援 SPI，必須將 **CS** 接到 **3.3V** 才能告訴晶片要用 I2C 通訊。
> * **裝置地址 (Device Address)**：將 **SDO** 接到 **GND**，其 7-bit 地址為 `0x53`；若接到 **3.3V**，地址會變為 `0x1D`。在 STM32 HAL 庫中，記得要左移一位 (`0x53 << 1`) 變成 `0xA6`。
